# Public static assets

Files in this folder are served from the site root.

- **`/images/`** — Logos and other images (e.g. brand assets).
  - **L_01.png** — Flow Productions logo (circular, black on white). Used on login, dashboard, onboarding, and admin.
  - **L_02.png** — Flow Productions logo (white on black). Use for dark backgrounds or as alternate.
  - Reference in app: `src="/images/L_01.png"` or `src="/images/L_02.png"`.
